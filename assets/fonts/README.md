# Fuente BebasNeue

Para que el splash screen funcione correctamente, necesitas descargar la fuente Bebas Neue.

## Pasos:

1. Descarga la fuente desde: https://fonts.google.com/specimen/Bebas+Neue
2. Extrae el archivo `BebasNeue-Regular.ttf`
3. Col√≥calo en esta carpeta: `assets/fonts/BebasNeue-Regular.ttf`
4. Ejecuta `flutter pub get`

## Alternativa:

Si prefieres usar otra fuente similar a BBH Bartle, puedes:
- Usar `Bebas Neue` (gratuita)
- Usar `Oswald` (gratuita)
- Usar `Anton` (gratuita)

Simplemente descarga el archivo .ttf y reemplaza el nombre en `pubspec.yaml` y en `splash_screen.dart`.
